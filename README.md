# 🧮 SmartFractionPlus

**SmartFractionPlus** is a lightweight Python class that implements fraction arithmetic and parsing entirely from scratch — no external libraries, no shortcuts. It’s designed to showcase clean object-oriented design and deep understanding of number theory fundamentals.

---

## 🚀 Features

- ✅ Arithmetic operations: `+`, `-`, `*`, `/`
- ✅ Comparison operators: `==`, `<`, `>`, `<=`, `>=`
- ✅ Auto-simplification and sign normalization
- ✅ Conversion from:
  - String format (`"3/4"`, `"1 1/2"`)
  - Decimal values (`0.75`)
- ✅ Conversion to:
  - Float (`to_float()`)
  - Integer (`to_int()`)
  - String (`to_str()`)
- ✅ Hashable and safe for use in sets/dictionaries

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Design:** Object-Oriented Programming
- **No Dependencies:** 100% custom logic — no use of `fractions.Fraction` or external math libraries
